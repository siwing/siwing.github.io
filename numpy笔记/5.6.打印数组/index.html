    5.6.打印数组

    import numpy
import numpy as np

    打印数组

    当打印ndarray时，numpy按照Python的嵌套list的格式打印输出，但是按照以下顺序打印：

    1. 最底层的axis按照从左到右的顺序输出

    2. 次底层的axis按照从上到下的顺序输出

    3. 其他层的axis也是按照从上到下的顺序输出，但是每个slice中间间隔一条空行

    如： 一维的ndarray按行打印；二维的ndarray按照矩阵打印；三维的ndarray按照矩阵的list打印

    如果ndarray太大，那么numpy默认跳过中间部分的数据而只是输出四个角落的数据。

    要想任何时候都打印全部数据，可以在print(array)之前设置选项

    numpy.set_printoptions(threshold=None)

    这样后续的打印ndarray就不会省略中间数据。

    np.set_printoptions(threshold=5)
np.arange(200)

    array([  0,   1,   2, ..., 197, 198, 199])
